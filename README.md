# Биржа проектов для студентов

## Для кого создается продукт

Для студентов/программистов, которым негде получать практический опыт, ведь стажировки требуют опыта. 
Для компаний, которые тратят время на поиск стажёров, но не могут проверить навыки кандидатов

## Чем продукт поможет

Можно написать платформу, на которой компании будут публиковать мини-проекты, а команды/соло разработчики браться за их выполнение

- **Для разработчика** это реальный опыт + портфолио + шанс на стажировку
- **Для компании** это готовая задача + отобранные кандидаты

## Стек

- Backend: Django
- Frontend: Django Templates
- DB: PostgreSQL
- Docker + Docker Compose

## API документация

страница Swagger
`http://localhost:8000/api/swagger/`  

документация Redoc
`http://localhost:8000/api/redoc/`

## Автозаполнение БД

заполнить
`docker-compose exec web python manage.py populate_db`

очистить
`docker-compose exec web python manage.py populate_db --clear`

## Докер / поднять проект

собрать проект
`docker compose up --build` 

очистить контейнеры 
`docker compose down -v`
